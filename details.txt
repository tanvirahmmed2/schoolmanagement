# SCHOOL SAAS PLATFORM — MAIN WEBSITE DETAILS

## Overview

A modern multi-tenant School Management SaaS platform built using:

* Next.js
* Next.js App Router
* Next.js API Routes
* JavaScript
* PostgreSQL
* Tailwind CSS

The platform allows educational institutions to:

* Register accounts
* Purchase subscription plans
* Submit payment information
* Get institution approval from managers
* Receive hosted school management systems

This platform works similarly to:

* Shopify
* Google Workspace
* Notion

But specifically for:

* Schools
* Colleges
* Coaching Centers
* Madrasas
* Educational Institutions

# 5. PURCHASE APPROVAL SYSTEM

## Purchase Flow

### Step 1

Client selects plan

### Step 2

Client submits:

* bKash transaction ID
* payment amount

### Step 3

Purchase request goes to manager panel

### Step 4

Manager can:

* Approve
* Reject
* Delete request

### Step 5 (IMPORTANT)

If approved:

* subscription created
* tenant created
* hosted institution activated
* support ticket automatically created between manager and client
